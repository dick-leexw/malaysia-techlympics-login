# Malaysia Techlympics 2026 Login Prototype

A responsive React + Tailwind CSS login page prototype for Malaysia Techlympics 2026.

## Run Locally

Use any local static server, or run the included Node server:

```powershell
npm start
```

Then open:

```text
http://localhost:5173
```

This prototype uses CDN-hosted React, Babel, and Tailwind CSS, so no backend or install step is required.
