# Malaysia Techlympics 2026 Login Prototype

A responsive React + Tailwind CSS login page prototype for Malaysia Techlympics 2026.

## Run Locally

Install dependencies, then run the Vite dev server:

```powershell
npm install
npm start
```

Then open:

```text
http://localhost:5173
```

This prototype uses React, Vite, and Tailwind CSS. No backend is connected yet.

## Deploy to Vercel

Upload the files at the project root, not the outer folder itself:

```text
index.html
package.json
README.md
postcss.config.js
tailwind.config.js
vite.config.js
vercel.json
src/App.jsx
src/index.css
src/main.jsx
```

In Vercel, use:

```text
Framework Preset: Vite
Build Command: npm run build
Output Directory: dist
Install Command: npm install
```
